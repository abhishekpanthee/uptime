\chapter{PROJECT TIMELINE}

The project will be executed over a period of approximately 3 months, divided into several key phases. Each phase includes specific tasks and milestones to ensure steady progress toward project completion.

\section{Project Schedule Overview}

The project timeline spans from December 12, 2025 to March 31, 2026, covering the following major phases:

\begin{enumerate}
    \item \textbf{Project Initiation (Dec 12 - Dec 20):} Topic selection, problem definition, and literature review
    \item \textbf{System Design (Dec 18 - Dec 30):} Requirements analysis, database design, and architecture design
    \item \textbf{Proposal Preparation (Dec 28 - Jan 2):} Proposal writing and submission
    \item \textbf{Proposal Submission:} January 2, 2026
    \item \textbf{Development (Jan 3 - Mar 10):} Backend development, frontend development, monitoring service implementation, and system integration
    \item \textbf{Testing \& Documentation (Mar 5 - Mar 28):} Testing, bug fixes, deployment, and final report preparation
    \item \textbf{Final Defense:} March 31, 2026
\end{enumerate}
