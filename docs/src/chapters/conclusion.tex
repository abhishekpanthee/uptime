\chapter{CONCLUSION}

\section{Summary}

This proposal presents the Uptime Tracker, a web-based monitoring system for tracking website availability and response times. The system uses Next.js frontend, Elysia.js backend, and PostgreSQL database on Bun runtime.

\section{Expected Contributions}

The project will deliver: a functional web application with 5-minute monitoring intervals, PostgreSQL database for analytics, interactive dashboard with status visualization, public status pages, and complete documentation.

\section{Feasibility}

\begin{itemize}
    \item \textbf{Technical:} All technologies are well-documented and actively maintained
    \item \textbf{Time:} 3-month timeline with buffer for challenges
    \item \textbf{Resource:} Open-source tools with free-tier cloud options
    \item \textbf{Scope:} Realistic for a DBMS academic project
\end{itemize}

\section{Conclusion}

The Uptime Tracker represents a practical application of database management combined with modern web development. The project team is confident in successful completion within the proposed timeline, delivering a functional website monitoring system for Thapathali Campus.
